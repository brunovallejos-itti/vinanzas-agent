# Agente Vinanzas
Agente IA para vinanzas en donde se podran encontrar varias informaciones sobre las politicas de la empresa, como tambien datos necesarios para la consulta del estado de los clientes. Segun informes en PDF.

## Como instalar el Agente

Desde el OCI es necesario realizar la instalación de los paquetes
mencionados en el archivo **requirements.txt**

### Parametros para la instalación

> GOOGLE_API_KEY=*tu_api_key*

## Como utilizar el Agente

Se genera una URL que es necesario visitarla para acceder al agente.  

Podrá consultar información sobre los archivos PDF colocados dentro de la carpeta **PDFS** o adjuntados al Agente.
